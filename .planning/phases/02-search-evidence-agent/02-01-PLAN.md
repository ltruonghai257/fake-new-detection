# Plan 02-01: search_agent.py — Tier-Separated Queries and Evidence Graph Integration

**Goal:** Rewrite `search_agent()` to run 3 tier-separated query passes (trusted-domain, flagged-domain, unrestricted), tag every result via `classify_domain()`, merge+dedup+sort+cap the evidence list, call `EvidenceGraph.build_from_evidence()`, and write both `state["evidence_graph"]` and `state["evidence"]` for backward compatibility.

## must_haves

- `search_agent()` runs 3 passes: `include_domains=trusted_list`, `include_domains=flagged_list`, unrestricted
- Queries drafted once via `_draft_queries()`, reused across all 3 passes unchanged
- Every `Evidence` item has `source_tier` set by `classify_domain(url)` on the actual result URL
- Dedup by URL; first occurrence across all 3 passes wins
- Merge sorted by tier priority (trusted=0, flagged=1, social=2, unknown=3) then score desc within tier; cap at `2 × settings.max_results`
- `EvidenceGraph.build_from_evidence(evidence)` called on final capped list
- Return dict contains both `"evidence_graph"` (EvidenceGraph instance) and `"evidence"` (list, backward compat)
- Existing `pytest tests/` passes without regression

```python
def search_agent(state: FactCheckState) -> dict:
    statement = state["statement"]
    queries = _draft_queries(statement)

    trusted_list = [d.strip() for d in settings.trusted_domains.split(",") if d.strip()]
    flagged_list = [d.strip() for d in settings.flagged_domains.split(",") if d.strip()]

    seen: set = set()
    raw: List[Evidence] = []

    for include_domains in (trusted_list, flagged_list, None):
        for q in queries:
            results = web_search(q, include_domains=include_domains)
            for e in results:
                url = e.get("url", "")
                if url and url in seen:
                    continue
                if url:
                    seen.add(url)
                e["source_tier"] = classify_domain(url) if url else "unknown"
                raw.append(e)

    _tier_priority = {"trusted": 0, "flagged": 1, "social": 2, "unknown": 3}
    raw.sort(key=lambda e: (_tier_priority.get(e.get("source_tier", "unknown"), 3), -e.get("score", 0.0)))
    cap = 2 * settings.max_results
    evidence = raw[:cap]

    evidence_graph = EvidenceGraph.build_from_evidence(evidence)

    msg = f"[Search] {len(queries)} queries x 3 passes -> {len(evidence)} evidence items (capped at {cap})"
    if not settings.has_search():
        msg += " (no search provider configured; continuing on model output only)"

    return {
        "search_queries": queries,
        "evidence": evidence,
        "evidence_graph": evidence_graph,
        "messages": [("assistant", msg)],
    }
```
