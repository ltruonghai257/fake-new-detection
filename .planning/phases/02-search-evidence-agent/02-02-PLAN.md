## Task 02-02-T2: Add `include_domains` to `web_search.py`

<task>
<id>02-02-T2</id>
<wave>1</wave>
<type>execute</type>
<action>
Edit `factcheck_agents/tools/web_search.py`. Three changes:

**1. `_search_tavily()` signature + payload:**

Change signature from:
```python
def _search_tavily(query: str, max_results: int) -> List[Evidence]:
```
to:
```python
def _search_tavily(query: str, max_results: int, include_domains: list | None = None) -> List[Evidence]:
```

In the `json={...}` dict inside `requests.post(...)`, add the `include_domains` field conditionally. The cleanest approach is to build the payload dict first, then conditionally add the key:

```python
payload = {
    "api_key": settings.tavily_api_key,
    "query": query,
    "max_results": max_results,
    "search_depth": "advanced",
    "include_answer": False,
}
if include_domains:
    payload["include_domains"] = include_domains
resp = requests.post(
    "https://api.tavily.com/search",
    json=payload,
    timeout=30,
)
```

**2. `_search_google_cse()` signature + query filter:**

Change signature from:
```python
def _search_google_cse(query: str, max_results: int) -> List[Evidence]:
```
to:
```python
def _search_google_cse(query: str, max_results: int, include_domains: list | None = None) -> List[Evidence]:
```

Build the effective query before the request:
```python
effective_query = query
if include_domains:
    site_filter = " OR ".join(f"site:{d}" for d in include_domains)
    effective_query = f"{query} {site_filter}"
```

Replace `"q": query` with `"q": effective_query` in the `params` dict.

**3. `web_search()` signature + forwarding:**

Change signature from:
```python
def web_search(query: str, max_results: int | None = None) -> List[Evidence]:
```
to:
```python
def web_search(query: str, max_results: int | None = None, include_domains: list | None = None) -> List[Evidence]:
```

Forward `include_domains` to both provider calls:
- `results = _search_tavily(query, n, include_domains=include_domains)`
- `return _search_google_cse(query, n, include_domains=include_domains)`
</action>
