<action>
Create `factcheck_agents/graph_utils.py` with the `EvidenceGraph` class:

- Module docstring: "In-memory evidence graph backed by networkx.DiGraph."
- Imports: `from __future__ import annotations`, `import networkx as nx`, `from typing import Any, Dict, List, Optional`
- `EvidenceGraph` class:
  - `__init__(self) -> None`: creates `self.graph: nx.DiGraph = nx.DiGraph()`
  - `build_from_evidence(cls, evidence_list: List[Dict[str, Any]]) -> "EvidenceGraph"` (classmethod):
    - Creates empty `EvidenceGraph`
    - Adds `"statement"` node with `node_type="statement"`
    - For each item in `evidence_list`: node_id = `item.get("url") or str(hash(item.get("snippet", "")))`; adds node with `node_type="evidence"`, `title=item.get("title", "")`, `snippet=item.get("snippet", "")`, `source_tier=item.get("source_tier", "unknown")`; adds edge `("statement", node_id, type="mentions")`
    - Returns the instance
  - `add_node(self, id: Any, attrs: Dict[str, Any]) -> None`: calls `self.graph.add_node(id, **attrs)`
  - `add_edge(self, src: Any, dst: Any, type: str, attrs: Optional[Dict[str, Any]] = None) -> None`: calls `self.graph.add_edge(src, dst, type=type, **(attrs or {}))`
  - `to_evidence_list(self) -> List[Dict[str, Any]]`: iterates `self.graph.nodes(data=True)`, collects dicts for nodes where `data.get("node_type") == "evidence"`, returns list with keys `url`, `title`, `snippet`, `source_tier` — where `url` is the node id
</action>

Create `factcheck_agents/source_tier.py` with the `classify_domain()` function:

- Module docstring: "URL-to-source-tier classifier. Returns 'trusted', 'flagged', or 'unknown'."
- Imports: `from __future__ import annotations`, `from urllib.parse import urlparse`, `from .config import settings`
- Function `classify_domain(url: str) -> str`:
  - Parse: `hostname = urlparse(url).hostname or ""`
  - Normalize: `domain = hostname.removeprefix("www.")`
  - Build lists: `trusted = [d.strip() for d in settings.trusted_domains.split(",") if d.strip()]`; same for `flagged`
  - Trusted check: `if any(domain == t or domain.endswith("." + t) for t in trusted): return "trusted"`
  - Flagged check: same pattern with `flagged` list; `return "flagged"`
  - Fallback: `return "unknown"`
  - Empty/invalid URL (`domain == ""`) falls through to `return "unknown"` naturally
</action>

```
import pytest
from factcheck_agents.graph_utils import EvidenceGraph

def test_empty_evidence_creates_statement_node():
    eg = EvidenceGraph.build_from_evidence([])
    assert "statement" in eg.graph.nodes
    assert eg.graph.nodes["statement"]["node_type"] == "statement"

def test_single_evidence_creates_two_nodes():
    ev = [{"url": "https://vnexpress.net/a", "snippet": "test", "source_tier": "trusted"}]
    eg = EvidenceGraph.build_from_evidence(ev)
    assert len(eg.graph.nodes) == 2

def test_evidence_node_attributes():
    ev = [{"url": "https://vnexpress.net/a", "title": "Title", "snippet": "Snip", "source_tier": "trusted"}]
    eg = EvidenceGraph.build_from_evidence(ev)
    node = eg.graph.nodes["https://vnexpress.net/a"]
    assert node["source_tier"] == "trusted"
    assert node["title"] == "Title"
    assert node["node_type"] == "evidence"

def test_default_edge_type_is_mentions():
    ev = [{"url": "https://vnexpress.net/a", "snippet": "s"}]
    eg = EvidenceGraph.build_from_evidence(ev)
    edges = list(eg.graph.edges(data=True))
    assert any(d.get("type") == "mentions" for _, _, d in edges)

def test_add_edge_with_custom_type():
    eg = EvidenceGraph()
    eg.add_node("a", {"node_type": "evidence"})
    eg.add_node("b", {"node_type": "evidence"})
    eg.add_edge("a", "b", type="supports")
    edge_data = eg.graph.edges["a", "b"]
    assert edge_data["type"] == "supports"

def test_to_evidence_list_returns_evidence_nodes():
    ev = [
        {"url": "https://vnexpress.net/a", "title": "T1", "snippet": "S1", "source_tier": "trusted"},
        {"url": "https://example.com/b", "title": "T2", "snippet": "S2", "source_tier": "unknown"},
    ]
    eg = EvidenceGraph.build_from_evidence(ev)
    result = eg.to_evidence_list()
    assert len(result) == 2
    urls = {item["url"] for item in result}
    assert "https://vnexpress.net/a" in urls
    assert "https://example.com/b" in urls
```
