## Milestone Summary

**Key Decisions:**
- Use ResNetCOOLANT (PatchedCOOLANT) as primary model — Adapts for arbitrary feature dims
- PhoBERT-base-v2 as default text encoder — Best Vietnamese NLP model
- HDF5 for preprocessed features — Efficient random access for DataLoader
- MLflow for experiment tracking — Low setup overhead; local tracking
- notebooks/ as target directory — Consolidates all experiment code

**Issues Resolved:**

- Config-driven approach implemented across all notebooks
- Reproducible end-to-end pipeline established
